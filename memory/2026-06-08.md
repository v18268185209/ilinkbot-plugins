## 2026-06-08 P0 改进完成记录

### 新增功能
1. **健康检查 API**
   - GET /api/wechathlink/admin/health/plugin - 插件全局健康检查
   - GET /api/wechathlink/admin/health/account?accountId={id} - 账号级健康检查
   - GET /api/wechathlink/admin/account/{id}/health - 账号详情健康检查

2. **轮询管理 API**
   - POST /api/wechathlink/admin/account/{id}/poller/start - 启动轮询
   - POST /api/wechathlink/admin/account/{id}/poller/stop - 停止轮询

3. **Webhook 自动重试**
   - WechathlinkWebhookRetryScheduler 每5分钟自动重试失败投递
   - 最多重试10次，超过后停止自动重试

4. **pom.xml 增强**
   - 添加 spring-boot-maven-plugin 支持独立模式
   - 添加 profiles: standalone, skip-frontend, prod

5. **编译修复**
   - defaultValue 方法签名修复 (2->3参数)
   - 新增 logMapper 注入
   - InterruptedException 处理
